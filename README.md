#### Requerimientos
```
npm
nodejs
mysql-server
```
s
- setup .env
```
PORT=
JWT_SECRET=
DB_USER=
DB_PASSWORD=
DB_NAME=
```
#### Instalacion
```
git clone https://github.com/RenzoQuispe/GestorTareasWeb.git
cd GestorTareasWeb
npm install
cd client
npm install
npm run dev
cd ..
npm run dev
```
Entonces:
```
WebApp en:   http://localhost:5173/
```
